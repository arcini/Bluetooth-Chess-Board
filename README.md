# Bluetooth-Chess-Board
This repository contains the firmware for an in-progress electronic chess board personal/side project. 

# Tooling And Devices

RTOS: Mbed OS 6

MCU: STM32WB55RG

MCU Development Board: ST NUCLEO-WB15CC
